# 🚀 Solo Technical Founder

> Your AI co-founder that never sleeps

A Vienna content profile for solo founders building and shipping products. Includes quick actions for changelog generation, launch marketing, revenue tracking, outreach, and landing page copy.

## What's included

- **Quick Actions** — Generate changelogs, draft launch tweets, check revenue, write outreach emails, create landing page copy
- **Skills** — Getting started guide for Vienna profiles
- **Plugins** — Linear for project management

## Install

Fork this profile in Vienna:

```
vienna://profile/github.com/tryvienna/profile-solo-founder
```

Or go to **Settings → Profiles → Fork** and paste:
```
https://github.com/tryvienna/profile-solo-founder.git
```

## Customize

After forking, this profile lives at `~/.vienna/profiles/solo-founder/`. Edit any file to make it yours — add quick actions, swap plugins, tweak skills.

## Share

Push your customized profile to your own GitHub repo and share the `vienna://profile/...` link. Others can fork your version in one click.
